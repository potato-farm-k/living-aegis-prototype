# Living Aegis Prototype

방어를 통해 반격 에너지를 모으는 2D 탑다운 우주 방어 슈팅 게임 프로토타입입니다.  
외부 라이브러리 없이 HTML, CSS, JavaScript와 Canvas API만 사용했습니다.

## 실행 방법

별도의 빌드 과정이나 패키지 설치가 필요하지 않습니다.

1. 저장소를 내려받습니다.
2. `index.html`을 브라우저에서 엽니다.

로컬 서버로 실행하려면 프로젝트 폴더에서 다음 명령을 사용할 수 있습니다.

```bash
python3 -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000`을 엽니다.

### GitHub Pages 배포

1. 이 프로젝트를 GitHub 저장소의 기본 브랜치에 푸시합니다.
2. 저장소의 **Settings → Pages**로 이동합니다.
3. **Deploy from a branch**를 선택하고 기본 브랜치의 `/ (root)`를 지정합니다.

정적 파일만 사용하므로 추가 설정 없이 GitHub Pages에서 실행됩니다.

## 조작법

| 입력 | 기능 |
| --- | --- |
| 마우스 | 화면 중앙의 플레이어를 360도 회전 |
| `WASD` | 화면 기준 상하좌우 방향으로 선회, 키 해제 시 현재 방향 유지 |
| 마우스 좌클릭 또는 `F` | 약한 생체 탄 연속 발사 |
| `Space` | 조준 방향으로 짧은 전방 부채꼴 방어막 전개 |
| `E` 또는 마우스 우클릭 | 방어 에너지가 가득 찼을 때 강한 생체 파동 발사 |
| `Enter` | Game Over 화면에서 재시작 |

## 주요 기능

- 별과 성운이 흐르는 Canvas 우주 배경
- 화면 중앙 근처에 고정된 플레이어와 방향 기반 무한 월드 스크롤
- 가까운 별과 먼 별의 속도 차이를 이용한 패럴랙스 비행 효과
- 원형 코어, 유기적인 외피, 촉수 형태로 표현한 생체 메카닉 플레이어
- 돌진형 적과 거리를 유지하며 탄을 발사하는 사격형 적
- 전방 120도 부채꼴 방어막과 재사용 대기시간
- 적 탄 방어 성공 시 축적되는 방어 에너지
- 에너지 100%에서 사용할 수 있는 광역 생체 파동
- 생존 시간에 따라 빨라지는 적 생성과 난이도
- 시작 시 선택 가능한 비기너 모드: 화면의 적 기체를 최대 10기로 제한
- Game Over 화면에서 다음 게임의 비기너 모드 사용 여부 선택
- 우측 상단 HUD에서 현재 모드와 살아 있는 적 기체 수 확인
- 체력, 방어 에너지, 생존 시간, 점수 HUD
- 피격, 방어 성공, 적 파괴, 생체 파동 파티클 및 화면 흔들림 효과
- Web Audio API로 합성한 무기 발사, 생체 파동, 방어막, 적 파괴 효과음
- 60초 생존 목표 알림과 이후 무한 생존 플레이
- Game Over 및 즉시 재시작
