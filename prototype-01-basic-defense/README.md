# Prototype 01 - Basic Defense

상태: `scaffold only`

이 폴더는 `living-aegis-prototype`의 첫 번째 prototype 공간이다.

목적은 basic defense shooting loop를 이후 실험할 수 있는 독립 실행 공간을 준비하는 것이다.

현재 단계에서는 정적 placeholder만 있으며, 실제 gameplay 기능은 구현하지 않았다.

이 scaffold에는 다음 system이 아직 없다.

- aiming
- firing
- enemies
- score
- missiles
- HUD

메인 게임은 `living-aegis-origin`에서 관리하고, simulator는 `living-aegis-simulator`에서 관리한다.
